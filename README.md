## О проекте

Данный проект представляет собой комплексный статистический анализ экспертной оценки заведений китайской кухни в Пекине.

**Цель исследования**: провести экспертную оценку качества заведений китайской кухни, выявить ключевые критерии оценки, определить барьеры развития и оценить согласованность мнений экспертов.

## Задачи исследования

1. **Создание матрицы оценок** - формирование анкеты и сбор данных от 25 экспертов
2. **Визуализация результатов** - построение диаграмм и графиков распределения оценок
3. **Статистический анализ**- расчет описательной статистики по критериям оценки
4. **Анализ согласованности** - расчет коэффициента конкордации Кендалла
5. **Факторный анализ** - выявление скрытых факторов, влияющих на оценку
6. **Оптимизация экспертной группы** - удаление "плохих" экспертов и пересчет результатов

## Установка и запуск

### Предварительные требования
- Python 3.12 или выше
- pip (менеджер пакетов Python)

### Шаг 1: Клонирование репозитория
```bash
git clone https://github.com/ваш-логин/expert_evaluation_chinese_cuisine_beijing.git
cd expert_evaluation_chinese_cuisine_beijing
```

### Шаг 2: Создание виртуального окружения
```bash
python -m venv .venv
source .venv/bin/activate  # Для Linux/Mac
# или
.venv\Scripts\activate     # Для Windows
```

### Шаг 3: Установка зависимостей
```bash
pip install -r requirements.txt
```

### Шаг 4: Запуск анализа
```bash
# Создание матриц оценок
python src/create_ratings_matrix.py

# Полный анализ данных
python src/complete_analysis.py
```

## Методология

### Методы статистического анализа:
1. **Описательная статистика** - среднее, медиана, мода, стандартное отклонение
2. **Корреляционный анализ** - матрица корреляций Пирсона
3. **Анализ согласованности** - коэффициент конкордации Кендалла (W)
4. **Факторный анализ** - метод главных компонент (PCA)
5. **Критерий хи-квадрат** - проверка статистической значимости

### Используемые библиотеки:
- **Pandas** - обработка табличных данных
- **NumPy** - математические вычисления
- **Matplotlib/Seaborn** - визуализация данных
- **SciPy** - статистические тесты
- **Scikit-learn** - факторный анализ

## Ключевые результаты

### 1. Оценка критериев качества
- **Наивысшие оценки**: Скорость обслуживания (4.20), Технологическая интеграция (4.16)
- **Наинизшие оценки**: Вегетарианские опции (2.80), Гигиена и чистота (2.96)

### 2. Согласованность экспертов
- **Исходный коэффициент конкордации**: W = 0.030 (низкая согласованность)
- **После удаления "плохих" экспертов**: W = 0.108 (статистически значимо, p < 0.05)
- **Удалено экспертов**: 4 из 25

### 3. Факторный анализ
Выявлено 3 основных фактора, объясняющих 76.1% дисперсии:
1. **Фактор качества** (46.7%) - аутентичность, качество ингредиентов
2. **Фактор сервиса** (18.6%) - гигиена, атмосфера, технологии
3. **Фактор разнообразия** (10.8%) - вегетарианские опции, меню
